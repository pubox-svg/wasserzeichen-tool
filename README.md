# wasserzeichen-tool
Ermöglicht es vordefinierte Wasserzeichen in Bildern einzufügen. 
Benötigt zwei Dateien. A das Bild und B das Wasserzeichen als png mit transparentem Hintergrund.

Wie funktioniert's?
1. Bild und Wasserzeichen hochladen
2. Wasserzeichen mit der Maus oder der Hand verschieben.
3. Größe des Wasserzeichens gegebenenfalls anpasen.
4. Opazität einstellen.
5. Bild herunterladen und umbenennen
